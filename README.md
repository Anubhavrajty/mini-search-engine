# Mini Search Engine

A CLI Python search tool for Termux/Linux.

## Commands

- `g <query>` → Google search  
- `d <query>` → DuckDuckGo search  
- `y <query>` → YouTube search  
- `w <query>` → Wikipedia search  
- `gh <query>` → GitHub search  
- `so <query>` → Stack Overflow search  
- `img <query>` → Image search (Google Images)

## Example

```bash
search g python loops
search y AI documentary
search gh python projects
